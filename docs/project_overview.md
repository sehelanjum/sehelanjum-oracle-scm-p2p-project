# Oracle SCM P2P Project – Shumayla Textiles

## Objective
To simulate a real-world Procure-to-Pay (P2P) cycle using Oracle SCM.

## Business Scenario
Shumayla Textiles purchases raw materials from suppliers to manufacture garments.

## Process Flow
1. Supplier Creation
2. Item Creation
3. Purchase Order Creation
4. Goods Receipt
5. Invoice Creation
6. Payment Processing

## Tools Used
- Oracle SCM Cloud
- Excel
- GitHub
